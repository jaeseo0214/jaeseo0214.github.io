# 자바 스윙 컴포넌트 활용






## 1. 스윙 컴포넌트


자바에서 GUI 프로그래밍은<br>
스윙 컴포넌트를 이용하는 방법과<br>
직접 그래픽으로 화면을 구성하는 방법으로 나뉜다.<br>
이번 글에서는 스윙 컴포넌트를 이용한 GUI 프로그래밍 방법을 살펴본다.<br>

스윙에서 사용하는 대부분의 컴포넌트는<br>
JComponent를 상속받아 구현되어 있으며,<br>
JFrame과 같은 최상위 컨테이너는 예외이다.<br>

JComponent는 추상 클래스로,<br>
스윙 컴포넌트들이 공통으로 사용하는 메소드와 상수들을 제공한다.<br>

다음은 JComponent와 그 상위 클래스(Component, Container)에서<br>
제공하는 주요 메소드들이다.<br>

```java
// 컴포넌트의 모양과 관련된 메소드
void setForeground(Color)   // 전경색 설정
void setBackground(Color)   // 배경색 설정
void setOpaque(boolean)     // 불투명성 설정
void setFont(Font)          // 폰트 설정
Font getFont()              // 폰트 리턴
```


```java
// 컴포넌트의 상태와 관련된 메소드
void setEnabled(boolean)    // 컴포넌트 활성화/비활성화
void setVisible(boolean)    // 컴포넌트 보이기/숨기기
boolean isVisible()          // 컴포넌트의 보이는 상태 리턴
```


```java
// 컴포넌트의 위치와 크기에 관련된 메소드
int getWidth()              // 폭 리턴    
int getHeight()             // 높이 리턴
int getX()                  // x 좌표 리턴
int getY()                  // y 좌표 리턴
Point getLocationOnScreen() // 스크린 좌표상에서의 컴포넌트 좌표
void setLocation(int, int)  // 위치 지정
void setSize(int, int)      // 크기 지정
```


```java
// 컨테이너를 위한 메소드
Component add(Component)        // 자식 컴포넌트 추가
void remove(Component)          // 자식 컴포넌트 제거
void removeAll()                // 모든 자식 컴포넌트 제거
Component[] getComponents()     // 자식 컴포넌트 배열 리턴
Container getParent()           // 부모 컨테이너 리턴
Container getTopLevelAncestor() // 최상위 부모 컨테이너 리턴
```






## 2. JLabel


JLabel은 문자열과 이미지를 출력할 때<br>
사용되는 레이블 컴포넌트이다.<br>


### JLabel 생성자


레이블의 생성자는 다음과 같다.<br>


```java
JLabel()    // 빈 레이블
JLabel(Icon image)  // 이미지 레이블
JLabel(String text) // 문자열 레이블
JLabel(String text, Icon image, int hAlign) // 문자열과 이미지 모두 가진 레이블

* hAlign: 텍스트와 이미지의 수평 정렬 값으로 SwingConstants.LEFT, SwingConstants.RIGHT, SwingConstants.CENTER 중 하나
```


이미지 레이블을 생성할 때는 이미지 파일의 경로명을 사용한다.<br>
경로명이 "images/img.jpg" 라면,<br>
img.jpg는 프로그램의 실행 기준 디렉터리에서<br>
images 폴더 안에 위치해야 한다.<br>
실행 기준 디렉터리는 일반적으로 프로젝트 루트 디렉터리이지만,<br>
실행 환경에 따라 달라질 수 있다.<br>

다음은 JLabel을 활용한 예제이다<br>


```java
public class JLabelEx extends JFrame {

	public JLabelEx() {
		super("JLabel 예제");
		setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

		Container c = getContentPane();

		c.setLayout(new FlowLayout());

		// 문자열 레이블
		JLabel textLa = new JLabel("귀여운 강아지");

		// 이미지 레이블
		ImageIcon img = new ImageIcon("images/dog.jpg");
		JLabel imgLa = new JLabel(img);

		// 문자열과 이미지 레이블
		ImageIcon img2 = new ImageIcon("images/cat.jpg");
		JLabel imgTextLa = new JLabel("야옹", img2, SwingConstants.CENTER);

		// 컨텐트팬에 모두 삽입
		c.add(textLa);
		c.add(imgLa);
		c.add(imgTextLa);

		setSize(300, 500);
		setVisible(true);
	}

	public static void main(String[] args) {
		new JLabelEx();
	}

}
```

## 3. JButton


JButton은 사용자의 동작을 입력받기 위해 사용하는 버튼 컴포넌트이며,<br>
버튼이 클릭되면 ActionEvent가 발생한다.<br>


### JButton 생성자


버튼의 생성자는 다음과 같다.<br>


```java
JButton()   // 빈 버튼
JButton(Icon image) // 이미지 버튼
JButton(String text)    // 문자열 버튼
JButton(String text, Icon image)    // 문자열과 이미지 버튼
```


JButton은 마우스 상태에 따라 서로 다른 아이콘을 설정할 수 있다.<br>


### normalIcon


버튼이 기본 상태에 있을 때 출력되는 아이콘이다.<br>
생성자나 JButton의 setIcon(Icon image)을 통해 설정한다.<br>


### rolloverIcon


마우스 포인터가 버튼 위에 올라갔을 때 출력되는 아이콘이다.<br>
JButton의 setRolloverIcon(Icon image)을 통해 설정한다.<br>


### pressedIcon


마우스 버튼이 눌러져 있는 동안 출력되는 아이콘이다.<br>
JButton의 setPressedIcon(Icon image)을 통해 설정한다.<br>


이 세 가지 아이콘을 JButton에 설정하면,<br>
사용자의 마우스 동작에 따라<br>
자동으로 버튼 이미지가 변경되어 출력된다.<br>

다음은 JButton을 활용한 예제이다.<br>


```java
public class JButtonEx extends JFrame {

	public JButtonEx() {
		super("JButton 예제");
		setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

		Container c = getContentPane();

		c.setLayout(new FlowLayout());

		// 3개의 이미지를 읽어들여 아이콘을 만듦
		ImageIcon normalIcon = new ImageIcon("images/normal.jpg");
		ImageIcon rolloverIcon = new ImageIcon("images/rollover.jpg");
		ImageIcon pressedIcon = new ImageIcon("images/pressed.jpg");

		// 3개의 아이콘을 가진 버튼 컴포넌트 생성
		JButton btn = new JButton("BBB", normalIcon);
		btn.setRolloverIcon(rolloverIcon);
		btn.setPressedIcon(pressedIcon);

		c.add(btn);

		setSize(500, 350);
		setVisible(true);
	}

	public static void main(String[] args) {
		new JButtonEx();
	}
}
```

JLabel에서 이미지 레이블을 만들 때와 마찬가지로<br>
이미지는 프로그램의 실행 기준 디렉터리 하위의<br> 
images 폴더에 위치해야 한다.<br>

## 4. JCheckBox


JCheckBox는 선택과 비선택의 두 상태를 가지는<br>
체크박스를 만들기 위한 컴포넌트이다.<br>


### JCheckBox 생성자


체크박스의 생성자는 다음과 같으며,<br>
selected 값을 지정하지 않으면 기본적으로 비선택 상태로 생성된다.<br>


```java
JCheckBox() // 빈 체크박스
JCheckBox(Icon image)   // 이미지 체크박스
JCheckBox(Icon image, boolean selected) // 이미지 체크박스
JCheckBox(String text, Icon image)  // 문자열과 이미지 체크박스
JCheckBox(String text, Icon image, boolean selected)    // 문자열과 이미지 체크박스

* selected: true 이면 선택 상태로 초기화
```


### ItemEvent 처리


체크박스에 ItemEvent 처리를 하는 것도 가능하다.<br>
ItemEvent는 체크박스가 선택되거나 해제될 때 발생하는 이벤트이다.<br>
그러나 선택 상태가 변경되지 않으면 ItemEvent는 발생하지 않는다.<br>

체크박스의 선택/해제는 주로 사용자의 마우스나 키보드로 이루어진다.<br>
하지만 프로그램에서 JCheckBox 객체의 setSelected() 메소드를 사용하여<br>
선택 상태를 변경할 수도 있다.<br>


```java
JCheckBox c = new JCheckBox("체크");
c.setSelected(true);    // true면 선택, false면 해제
```


두 가지 경우 모두 ItemEvent가 발생한다.<br>
ItemEvent는 ItemListener에 의해 처리되며,<br>
ItemListener 인터페이스에는 itemStateChanged() 메소드가 선언되어 있다.<br>


```java
void itemStateChanged(ItemEvent e)  // 체크박스의 선택 상태가 변하는 경우 호출
```


ItemEvent가 발생하면 ItemEvent 객체가 생성되어,<br>
위의 메소드에 전달된다.<br>
ItemEvent 객체는 다음 메소드를 통해<br>
체크 상태의 변화와 이벤트 소스를 제공한다.<br>


```java
int getStateChange()
// 체크박스가 선택된 경우 ItemEvent.SELECTED, 해제된 경우 ItemEvent.DESELECTED 리턴

Object getItem()
// 이벤트를 발생시킨 객체 리턴 (JCheckBox의 경우 JCheckBox 객체)
```


### 리스너 등록


체크박스 컴포넌트에 리스너를 등록하는 방법은 다음과 같다.<br>


```java
checkbox.addItemListener(new MyItemListener());
```


다음은 JCheckBox를 활용한 예제이다.<br>


```java
public class JCheckBoxEx extends JFrame {

	private JCheckBox[] fruits = new JCheckBox[3]; // 체크박스 배열
	private String[] names = { "바나나", "포도", "메론" }; // 체크박스 문자열로 사용할 문자열 배열

	private JLabel sumLabel; // 계산 합을 출력할 레이블

	public JCheckBoxEx() {
		super("JCheckBox 예제");
		setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

		Container c = getContentPane();

		c.setLayout(new FlowLayout());

		JLabel la = new JLabel("바나나: 100원, 포도: 300원, 메론: 700원");
		c.add(la);

		// 3개의 체크박스 컴포넌트 생성, 컨텐트팬에 삽입, Item 리스너 등록
		MyItemListener listener = new MyItemListener();

		for (int i = 0; i < fruits.length; i++) {
			fruits[i] = new JCheckBox(names[i]);
			fruits[i].setBorderPainted(true); // 체크박스 외곽선 설정

			c.add(fruits[i]);
			fruits[i].addItemListener(listener);
		}

		sumLabel = new JLabel("현재 가격: 0원");

		c.add(sumLabel);

		setSize(250, 200);
		setVisible(true);
	}

	// Item 리스너 구현
	class MyItemListener implements ItemListener {

		private int sum = 0;

		// 체크박스의 선택 상태가 변하면 itemStateChanged() 호출
		public void itemStateChanged(ItemEvent e) {

			// 체크 박스가 선택된 경우
			if (e.getStateChange() == ItemEvent.SELECTED) {
				if (e.getItem() == fruits[0]) {
					sum += 100;
				} else if (e.getItem() == fruits[1]) {
					sum += 300;
				} else {
					sum += 700;
				}
			}

			// 체크박스가 해제된 경우
			else {
				if (e.getItem() == fruits[0]) {
					sum -= 100;
				} else if (e.getItem() == fruits[1]) {
					sum -= 300;
				} else {
					sum -= 700;
				}
			}
			sumLabel.setText("현재 가격: " + sum + "원"); // 합 출력
		}
	}

	public static void main(String[] args) {
		new JCheckBoxEx();
	}
}
```

## 5. JRadioButton


JRadioButton은 체크박스와 유사하게<br>
선택과 비선택의 두 가지 상태를 가지는<br>
라디오 버튼 컴포넌트이다.<br>

체크박스와의 가장 큰 차이점은 다음과 같다.<br>

체크박스는 각 항목이 서로 독립적으로 선택/해제가 가능하다.<br>

라디오버튼은 같은 버튼 그룹에 속한<br>
여러 버튼 중 오직 하나만 선택할 수 있다.<br>

즉, 라디오버튼은 상호 배타적인 선택을 표현할 때 사용된다.<br>


### JRadioButton 생성자


라디오버튼의 생성자는 다음과 같으며,<br>
selected 값을 지정하지 않으면 기본적으로 비선택 상태로 생성된다.<br>


```java
JRadioButton()                              // 텍스트와 이미지가 없는 라디오버튼
JRadioButton(Icon image)                   // 이미지 라디오버튼
JRadioButton(Icon image, boolean selected) // 이미지 라디오버튼
JRadioButton(String text)                  // 문자열 라디오버튼
JRadioButton(String text, boolean selected)// 문자열 라디오버튼
JRadioButton(String text, Icon image)      // 문자열과 이미지 라디오버튼
JRadioButton(String text, Icon image, boolean selected) // 문자열과 이미지 라디오버튼

* selected: true이면 선택 상태로 초기화됨
```


라디오버튼은 단순히 생성자만 호출하는 것 외에,<br>
다음과 같은 네 단계의 과정을 거쳐 사용한다.<br>


```java
// 1. 버튼 그룹 객체 생성
ButtonGroup group = new ButtonGroup();

// 2. 라디오버튼 컴포넌트 생성
JRadioButton banana = new JRadioButton("바나나");
JRadioButton grape  = new JRadioButton("포도");
JRadioButton melon  = new JRadioButton("메론");

// 3. 라디오버튼을 버튼 그룹에 추가
group.add(banana);
group.add(grape);
group.add(melon);

// 4. 라디오버튼을 컨테이너에 추가
container.add(banana);
container.add(grape);
container.add(melon);
```


⚠ 주의<br>
라디오버튼을 버튼 그룹에 추가하지 않으면<br>
라디오버튼이라 하더라도 체크박스처럼 중복 선택이 가능해진다.<br>


### ItemEvent 처리


라디오버튼 역시 선택되거나 해제되어 상태가 변경되면<br>
ItemEvent가 발생한다.<br>

라디오버튼의 선택/해제는 일반적으로<br>
사용자의 마우스 클릭 또는 키보드 입력을 통해 이루어지지만,<br>
프로그램 내부에서 setSelected() 메소드를 사용하여<br>
선택 상태를 변경할 수도 있다.<br>

ItemEvent의 처리 방식은<br>
JCheckBox 와 동일한 개념으로 이루어진다.<br>


### ItemEvent 발생 시 주의사항


라디오버튼 사용 시 주의할 점이 하나 있다.<br>

예를 들어,<br>
3개의 라디오버튼 A, B, C 가 같은 버튼 그룹에 속해 있고,<br>
이미 A가 선택된 상태에서 B를 선택하면<br>
ItemEvent는 총 두 번 발생한다.<br>

라디오버튼 A : 선택 해제(DESELECTED) 이벤트 발생<br>

라디오버튼 B : 선택(SELECTED) 이벤트 발생<br>

이는 버튼 그룹의 특성상<br>
기존에 선택된 버튼이 자동으로 해제되기 때문이며,<br>
정상적인 동작이다.<br>

다음은 JRadioButton을 활용한 예제이다.<br>


```java
public class JRadioButtonEx extends JFrame {

	public JRadioButtonEx() {
		super("JRadioButton 예제");
		setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

		Container c = getContentPane();

		c.setLayout(new FlowLayout());

		ButtonGroup g = new ButtonGroup(); // 버튼 그룹 객체 생성

		// 라디오버튼 3개 생성
		JRadioButton banana = new JRadioButton("바나나");
		JRadioButton grape = new JRadioButton("포도");

		g.add(banana);

		c.add(banana);
```

## 6. JTextField


JTextField를 이용하면<br>
한 줄의 문자열을 입력받는 텍스트 입력 필드를 만들 수 있다.<br>

텍스트필드에 문자열을 입력하던 중<br>
`<Enter>` 키를 누르면 ActionEvent가 발생한다.<br>

텍스트필드의 표시 크기(열의 개수)와<br>
초기 문자열은 응용프로그램에서 설정할 수 있다.<br>


### JTextField 생성자

텍스트필드의 생성자는 다음과 같다.<br>


```java
JTextField()	// 빈 텍스트필드
JTextField(int cols)	// 입력 창의 열의 개수가 cols개인 텍스트필드
JTextField(String text)	// text 문자열로 초기화된 텍스트필드
JTextField(String text, int cols)	// 입력 창의 열의 개수는 cols개, text 문자열로 초기화된 텍스트 필드
```


### 열(cols) 값에 대한 주의사항

열의 개수가 10이라는 의미는<br>
최대 10개의 문자를 입력할 수 있다는 뜻이 아니다.<br>

cols는 텍스트필드의 표시 폭을 의미하며,<br>
실제로 입력 가능한 문자 수는<br>
폰트의 크기와 문자 종류에 따라 달라진다.<br>

다음은 JTextField를 활용한 예제이다.<br>


```java
public class JTextFieldEx extends JFrame {

	public JTextFieldEx() {
		super("JTextField 예제");
		setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

		Container c = getContentPane();

		c.add(new JLabel("이름 "));
		c.add(new JTextField(20));

		c.add(new JTextField("경기도", 20));

		setSize(300, 150);
		setVisible(true);
	}

	public static void main(String[] args) {
		new JTextFieldEx();
	}
}
```

## 7. JTextArea


JTextArea를 이용하면<br>
여러 줄의 문자열을 입력받는 텍스트 입력 필드를 만들 수 있다.<br>

입력 필드의 표시 크기보다<br>
더 많은 줄의 문자열을 입력할 수 있지만,<br>
스크롤바를 사용하려면 JScrollPane에 삽입해야 한다.<br>

JTextArea 자체에는 스크롤 기능이 제공되지 않는다.<br>


### JTextArea 생성자


텍스트영역의 생성자는 다음과 같다.<br>


```java
JTextArea() 	// 빈 텍스트영역
JTextArea(int rows, int cols) 	// rows × cols 크기의 텍스트영역
JTextArea(String text) 	// text 문자열로 초기화된 텍스트영역
JTextArea(String text, int rows, int cols) 	// rows × cols 크기, text로 초기화된 텍스트영역
```


### JTextArea 컴포넌트 생성


텍스트영역을 생성하는 방법은 두 가지로 나뉜다.<br>


1️⃣ 스크롤바가 없는 텍스트영역<br>


```java
container.add(new JTextArea("Java", 7, 20));
```

이 경우 입력 내용이 표시 영역을 초과하더라도<br>
스크롤바는 제공되지 않는다.<br>


2️⃣ 스크롤바를 지원하는 텍스트영역<br>


```java
container.add(new JScrollPane(new JTextArea("Java", 7, 20)));
```


이 방법은 입력 내용이 지정된 크기를 초과하면<br>
자동으로 스크롤바가 나타나,<br>
내용을 스크롤하여 확인할 수 있다.<br>

다음은 JTextArea를 활용한 예제이다.<br>


```java
public class JTextAreaEx extends JFrame {

	private JTextField tf = new JTextField(20);
	private JTextArea ta = new JTextArea(7, 20);	// 7행 × 20열 크기의 텍스트영역
	public JTextAreaEx() {
		super("JTextArea 예제");
		setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

		Container c = getContentPane();

		c.setLayout(new FlowLayout());

		c.add(new JLabel("입력 후 <Enter> 키를 입력하세요."));
		c.add(tf);
		c.add(new JScrollPane(ta));

		// 텍스트필드에 Action 리스너를 등록
		tf.addActionListener(new ActionListener() {

			// <Enter> 키 입력 시 작동하는 리스너
			public void actionPerformed(ActionEvent e) {
				JTextField t = (JTextField) e.getSource();
				ta.append(t.getText() + "\n"); // 텍스트필드의 문자열을 텍스트영역에 추가
				t.setText(""); // 현재 텍스트필드에 입력된 내용 지우기
			}

		});

		setSize(300, 250);
		setVisible(true);
	}

	public static void main(String[] args) {
		new JTextAreaEx();
	}
}
```
